# How to contribute

## Setup steps

1. [Install pnpm](https://pnpm.io/installation#using-homebrew) if you don't have it installed. For example:
   ```shell
   brew install pnpm
   ```
2. Install the dependencies:
   ```shell
   pnpm install
   ```
3. Run the packages:
   ```shell
   pnpm run dev
   ```
