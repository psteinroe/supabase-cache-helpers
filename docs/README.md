# Documentation site

:point_right: [Check it out](https://supabase-cache-helpers.vercel.app) :point_left:

This package uses [nextra-theme-docs](https://nextra.site/docs/docs-theme/start) to generate documentation using MDX files.

## Search index

The search index is generated by [`pagefind](https://www.npmjs.com/package/pagefind).

An action runs after `next build` to generate the index in CI. You can also run the command below to generate it manually, but note that you need to run the `build` script first:

```shell
pnpm run generate-search-index
```
